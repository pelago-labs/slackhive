SlackHive ships with built-in starter personas for common roles (Data Analyst, Backend Engineer, Product Manager, Security Engineer, Technical Writer, and many more). You pick one in the Profile step of the new-agent flow and the agent starts with a persona, a system prompt, and a default skill set.

## Pick a persona

1. Open **New Agent** from the sidebar
2. Fill in the Identity step
3. On the Profile step, browse the persona list
4. Click the persona that fits
5. Create the agent, then tune details with Coach from the agent page

## What gets seeded

Each persona seeds:

- A **system prompt base** - role, tone, guardrails
- A **starter skill set** - domain conventions the agent always carries

The seeded content is research-backed, not generic filler. Personas are kept up to date in-tree.

## When to pick "Blank"

If none of the built-in personas fit your use case, pick **Blank**. You get an empty starting prompt and no seeded skills - Coach can walk you through building one from scratch.

## Customize after creation

The persona is a starting point, not a lock-in. After the agent is created:

- Edit the system prompt in **Instructions → System Prompt**
- Add or remove skills in **Instructions → Skills**
- Use [Coach](/agents/coach) to tune the identity interactively

<Note>
  For **boss agents**, the system prompt is locked. Boss identity is generated from the team registry and cannot be hand-edited. See [Boss Agents](/agents/boss-agents).
</Note>
